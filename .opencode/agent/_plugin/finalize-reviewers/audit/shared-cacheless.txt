# Process
1. Read `handoff_path`, `context_path`, and all `step_paths` from scratch.
2. Emit all current BLOCKING and ADVISORY findings.
3. Inspect all STEP items and required sections.
4. Apply rule cards in order: AUD-1 → AUD-2 → AUD-3 → AUD-4 → AUD-5 → AUD-6.
5. Return all findings directly in the output block. Do not write cache or actions files.
