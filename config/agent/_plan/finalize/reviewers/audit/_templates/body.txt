# Focus

## Read strategy
Trust step file diffs and handoff `## Settled Facts` for repo grounding. Only open source files for specific verification when a diff context line does not match or a finding needs exact confirmation.

Audit in order: fidelity → visibility → structure → completeness → economy → dead-code. Stop dead-code if no REMOVE steps.
