# Process
1. Read full artifacts
- Read the full `context_path` and relevant `draft_handoff_path` sections from scratch.

2. Inspect all items
- Inspect every `[P#]` item and required section before a READY handoff.

3. Inspect content
- Read `context_path` for all `[P#]` items.
- Check Open→Resolved transitions.
- On malformed-output retry without new Delta or Decision entries, reuse prior analysis and re-emit valid protocol output from the existing review state.

4. Emit findings inline
- Return all findings directly in the output block. Do not write cache or actions files.
