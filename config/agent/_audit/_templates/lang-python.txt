Enumerate: In `__all__` OR non-`_`-prefixed at module scope. Skip: `_`-prefixed.
Module Boundary: directory containing `__init__.py`.
Re-export: `__init__.py` re-imports `from .module import Name` → Name re-exported to package consumers.
Qualified Path: `from module import item`, `module.Item`, `module.function_name`.
Test Patterns: `test_*.py`, `tests/`.
Special Case: references in `__init__.py` re-exporting item from submodule within same package (`from .module import Name`) → **Internal non-test** (package's own API surface, not external demand). Real external consumers importing item from other packages still appear as External usages.
