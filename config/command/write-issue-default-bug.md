# Write Bug Report (Default Template)

Use this built-in template. Generate `ISSUE-[brief-description].md`.

```markdown
---
title: "[Bug Title]"
labels: [bug]
---

## Description
[Bug description]

## Steps to Reproduce
1. [Step 1]
2. [Step 2]

## Expected Behaviour
[What should happen]

## Actual Behaviour
[What actually happens]

## Additional Context
[Any additional information]
```
